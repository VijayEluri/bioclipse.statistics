Installing Bioclipse-R

Prerequisites

Bioclipse-R Integration is a set of plugins for Bioclipse. It's shipped without R and other necessary packages and tools, therefore you need to make sure that all the requirements are fulfilled before installing the feature in Bioclipse:

  1. R version 2.13 or higher
  2. Mac OS X users are highly recommended to install the XQuartz X11 for making the best use of the R graphical facilities
  3. Linux users need to:
    • install JDK (the tools included in JDK are necessary for installing the required R packages);
      if JDK is installed after R, run also 
      R CMD javareconf
    • set their own Library Search Paths entry for installing R packages; example for debian based distributions follows:
      run the following command in your terminal: 
      		grep R_LIBS_USER /etc/R/Renviron
      on Ununtu 64 bit the output should be: 
      		R_LIBS_USER=${R_LIBS_USER-'~/R/x86_64-pc-linux-gnu-library/2.14'}
		#R_LIBS_USER=${R_LIBS_USER-'~/Library/R/2.14/library'}
      create the directory path from the uncommented line: 
      		mkdir -p ~/R/x86_64-pc-linux-gnu-library/2.14

Download Bioclipse and use the Install New Bioclipse Feature option

This guarantees the latest version of Bioclipse and Bioclipse-R.

Bioclipse is available from www.bioclipse.net, see the installation page for more information.

After installing Bioclipse, start it and go to the menu Install > New Feature.... Check Bioclipse-R Integration and press finish, and after restart you have Bioclipse-R installed.
